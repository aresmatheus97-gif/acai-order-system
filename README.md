# Açaí Order System

A simple terminal-based ordering system developed in Python.  
This project simulates a basic ordering flow for an açaí shop.

## 📌 Features

- Choose size (P, M, G)
- Choose flavor (Açaí or Cupuaçu)
- Automatic price calculation
- Support for multiple items in a single order
- Input validation using nested loops (size and flavor)
- Improved input handling with `.upper()` and `.strip()` for consistency
- Delivery address input

## 🧠 Purpose

This project was created as a learning exercise to practice:

- Conditional statements (`if`, `elif`, `else`)
- Loops (`while`) - including nested loops for input validation
- User input handling
- Basic program flow control
- Accumulator pattern for calculations
