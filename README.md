# Açaí Order System

A simple terminal-based ordering system developed in Python.  
This project simulates a basic ordering flow for an açaí shop.

## 📌 Features

- Choose size (P, M, G)
- Choose flavor (Açaí or Cupuaçu)
- Automatic price calculation
- Option to add one extra item (complement)
- Input validation using loops
- Delivery address input

## 🧠 Purpose

This project was created as a learning exercise to practice:

- Conditional statements (`if`, `elif`, `else`)
- Loops (`while`)
- User input handling
- Basic program flow control

## 🚧 Status

This project is still in development and will be improved over time.

## 🚀 Future Improvements

- Use of lists and dictionaries for cleaner logic
